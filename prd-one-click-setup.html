PRD v4.0

One-Click Onboarding

Remove onboarding friction by eliminating pre-creation wizard steps. A single "Activate" click creates the account and issues a virtual card — security steps move to post-creation.

01 — What Changes & Why

What Changes & Why

Today's onboarding uses a multi-step wizard that collects SSN, enrolls 2FA, verifies password, reviews business info, and collects card + shipping details — all before the account is created. Every step loses users. We want to collapse this into a single "Activate" click.

The Problem: Onboarding Funnel Drop-off

Post-intent conversion: 3,025 clicked "Open Account" → 1,705 accounts created (56.4%) — 43.6% drop-off in the wizard

3

SSN and 2FA move from the wizard to join the existing password verification step. Fixed order: SSN → Password → 2FA. Every dashboard action is gated until security flow completes.

09 — Edge Cases

Edge Cases & Error States

ScenarioBehavior
Banking partner needs more info (KYC insufficient) "Awaiting Documents" state — same screen as today. User uploads required docs.
Application rejected Rejection screen — same as today. Contact support option.
Application pending review "Account Under Review" screen — same as today. Notification when resolved.
User already has a Checking account Skip activation — go straight to dashboard.
User closes browser during account creation Backend completes the request. On next visit, check capability status and show appropriate state.
User has existing Wix 2FA 2FA step is automatically skipped in the security flow. Still needs SSN + password.
User never completes security steps Account exists. Dashboard is viewable. Every action prompts the security flow. Banner persists.
User's password doesn't meet banking grade Password step prompts upgrade. Can't proceed to 2FA until password is compliant.
Physical card request fails Error in the request modal. Virtual card is unaffected. User can retry.
User freezes virtual card, then uses physical (or vice versa) Cards are independent. Freezing one doesn't affect the other.
